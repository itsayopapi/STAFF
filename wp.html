What is WP-CLI?

WP-CLI is the official command-line interface for WordPress.

It enables you to interact with your WordPress installation without using a web browser. Developers and system administrators use WP-CLI to:

  • Manage plugins, themes, and users
  • Perform database operations
  • Run custom scripts
  • Automate repetitive tasks
  • Manage multiple sites at once
Did you know? WP-CLI can process commands up to 10x faster than the WordPress admin dashboard and is used by all major WordPress hosting providers.

Essential WP-CLI Commands

The most frequently used commands in your workflow

System Information

wp --info

Displays information about your WP-CLI environment including PHP version, WP-CLI version, and WordPress installation details.

wp core version

Shows the currently installed WordPress version.

User Management

wp user list

Displays all users registered on your WordPress site.

wp user create username email@example.com --role=editor

Create a new user with specified role.

Plugin & Theme Management

wp plugin list

Lists all installed plugins with their status (active/inactive) and version.

wp plugin install akismet --activate

Install and activate a plugin from WordPress.org repository.

Database Operations

wp db optimize

Optimize the WordPress database tables.

wp db export backup.sql

Create a complete database backup with one command.

Advanced Features

Power tools for professional WordPress developers

Database Management

wp db query "SELECT * FROM wp_posts WHERE post_type='post';"

Run direct SQL queries on your WordPress database.

wp db export backup.sql --add-drop-table

Create a complete database backup with DROP TABLE statements.

Search & Replace

wp search-replace 'http://old.com' 'http://new.com' --all-tables

Safely replace URLs in your database (perfect for migrations).

wp search-replace 'foo' 'bar' --dry-run --verbose

Test replacements before actually making changes with detailed output.

Multisite Management

wp site list --field=url

List all sites in a WordPress Multisite network.

wp site create --slug=new-site --title="New Site"

Create a new site in a Multisite network.

Cron Management

wp cron event list

List all scheduled cron events.

wp cron event run hook_name

Run a specific cron event immediately.
