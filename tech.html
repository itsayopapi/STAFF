Framework-Specific Deployment Guides

Step-by-step instructions for deploying your favorite frameworks.

Next.js
Zero Config

Deploy a Next.js app with hybrid rendering (SSG/SSR/ISR). 

# Install CLI
npm install -g wefithost-cli

# Navigate to your project
cd my-next-app

# Deploy
wefithost deploy

# Automatically detects:
# - next.config.js for ISR/SSR
# - Image optimization
# - Edge functions

Your site will be live at https://your-project.wefithost.app

Django
Needs Config

Deploy a Django app with PostgreSQL and Redis caching. 

# Set environment variables
wefithost env:set DATABASE_URL=postgres://user:pass@host/db
wefithost env:set SECRET_KEY=your-secret-key
wefithost env:set DEBUG=False

# Add a Procfile
echo "web: gunicorn project.wsgi --bind 0.0.0.0:$PORT" > Procfile

# Deploy
wefithost deploy

Add requirements.txt for automatic dependency installation.

Laravel
Needs Config

Deploy a Laravel application with MySQL database. 

# Set environment variables
wefithost env:set DB_CONNECTION=mysql
wefithost env:set DB_HOST=your-db-host
wefithost env:set DB_DATABASE=your-db-name
wefithost env:set DB_USERNAME=your-db-user
wefithost env:set DB_PASSWORD=your-db-pass
wefithost env:set APP_KEY=base64:your-app-key

# Run migrations
wefithost run "php artisan migrate --force"

# Deploy
wefithost deploy

Make sure to include public/.htaccess for proper routing.

Ruby on Rails
Needs Config

Deploy a Rails application with PostgreSQL database. 

# Set environment variables
wefithost env:set DATABASE_URL=postgres://user:pass@host/db
wefithost env:set RAILS_ENV=production
wefithost env:set SECRET_KEY_BASE=your-secret-key

# Precompile assets
wefithost run "bundle exec rails assets:precompile"

# Run migrations
wefithost run "bundle exec rails db:migrate"

# Deploy
wefithost deploy

Make sure your Gemfile includes pg for PostgreSQL.

    Frequently Asked Questions

    Get answers to common questions about hosting with different technologies.

    How do I deploy a React application?

    Deploying a React app is simple with our platform. Just run wefithost deploy in your project directory. We automatically detect it's a React app, optimize the build, and deploy it with our global CDN. For Create React App, no additional configuration is needed.

    Can I use a custom domain with my project?

    Yes! You can connect any custom domain to your project. We provide free SSL certificates automatically and support apex domains (like example.com) and subdomains (like app.example.com). DNS configuration takes just a few minutes.

    How does database hosting work?

    We offer fully managed databases with automated backups, scaling, and high availability. You can provision a database with one click and connect it to your application using environment variables. We support PostgreSQL, MySQL, MongoDB, and Redis.

    What about environment variables?

    You can set environment variables through our dashboard or CLI. They're encrypted at rest and injected into your application at runtime. For sensitive values, we recommend using our secrets manager which provides additional security features.

    How do I handle background jobs?

    We support background workers and cron jobs. You can define worker processes in your Procfile or through our dashboard. For scheduled tasks, we offer a cron-like interface with minute-level precision.

    What about scaling?

    Our platform automatically scales your applications based on traffic. You can also set manual scaling rules or enable auto-scaling with custom metrics. For databases, scaling is as simple as moving a slider in our dashboard.
