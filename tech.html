Framework-Specific Deployment Guides

Next.js
Zero Config

Deploy a Next.js app with hybrid rendering (SSG/SSR/ISR).

# Install CLI
npm install -g your-cli

# Deploy
your-cli deploy --framework nextjs

Automatically detects next.config.js for ISR/SSR.

Django
Needs Config

Deploy a Django app with PostgreSQL and Redis caching.

# Set environment variables
your-cli env:set DATABASE_URL=postgres://user:pass@host/db

# Deploy with Gunicorn
your-cli deploy --procfile "web: gunicorn project.wsgi"

Add requirements.txt for automatic dependency installation.

Docker
Any Stack

Deploy custom containers with compose or Kubernetes.

# Example Dockerfile
FROM node:18
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
CMD ["npm", "start"]

Run your-cli deploy --docker from your project root.

Flask
Zero Config

Deploy a Python Flask API with automatic WSGI detection.

# app.py
from flask import Flask
app = Flask(__name__)

@app.route("/")
def hello():
    return "Hello World!"

Just run your-cli deploy in your project folder.
